# AI Accountability Coach

A personal high-performance execution coach powered by OpenAI.

## Features
- Daily automated planning emails
- Execution tracking via web dashboard
- Long-term performance feedback
- Calm but strict coaching style
- Time-blocked daily plans

## Tech Stack
- Python (Flask)
- OpenAI Assistants API
- SQLite (local)
- Gmail SMTP
- Google Cloud Run

python3 -m venv venv
source venv/bin/activate
